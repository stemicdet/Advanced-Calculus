    Section Week 2: The Mean Value Theorems (5.3)

    Subsection The Mean Value Theorem

    Imagine that you are going to sail to Ireland. You could choose to sail in a straight line path or you could choose a curvier route, but no matter what route you choose, at some point you have to head in the same direction as the straight line path.
    ðŸ”—
    This observation is expressed mathematically by a result known as the Mean Value Theorem, which states that for a differentiable function the slope of the secant line between \(x=a\) and \(x=b\) is equal to the slope of the tangent line at some point between \(a\) and \(b\)
    ðŸ”—

    Theorem 16. Mean Value Theorem.

    If \(f:[a,b] \to \mathbf{R}\) be continuous on \([a,b]\) and differentiable on \((a,b)\text{.}\) If \(f(a) = f(b)\text{,}\) then there exists a point \(c \in (a,b)\) where
    ðŸ”—
    \begin{equation*} f'(c) = \frac{f(b)-f(a)}{b-a} \end{equation*}
    ðŸ”—
    The importance of the Mean Value Theorem goes beyond its immediate geometric interpretation. The MVT is significant because it relates the value of the derivative of a function to the value of the function itself.
    ðŸ”—
    Several proofs of the Mean Value Theorem have been given. The first proof of the modern form of the MVT is due to Cauchy in 1823.
    ðŸ”—
    In 1882 Camille Jordan published a calculus textbook with an erroneous proof of the MVT. The error was pointed out by Giuseppe Peano, who noted in a letter to Jordan that his proof implicitly assumed the continuity of the derivative.
    ðŸ”—
    Peano also showed that Jordanâ€™s proof would not work for the function
    \begin{equation*} f(x) = \begin{cases} \hfill x^2 \sin(1/x) \amp \text{ if $x \neq 0$} \\ \hfill 0 \hfill \amp \text{ if $x =0 $} \\ \end{cases} \end{equation*}
    ðŸ”—
    We will prove the MVT by first proving the following special case first noted by Michel Rolle in 1690.
    ðŸ”—

    Theorem 17. Rolleâ€™s Theorem.

    Let \(f:[a,b]\to \mathbf{R}\) be continuous on \([a,b]\) and differentiable on \((a,b)\text{.}\) If \(f(a) = f(b)\text{,}\) then there exists a point \(c \in (a,b)\) such that \(f'(c) = 0\text{.}\)
    ðŸ”—
    ðŸ”—

    Proof.

    Since \(f\) is a continuous function on a compact set, \(f\) contains both a minimum and a maximum. If these occur at the endpoints, then the function is constant and we are done.
    ðŸ”—
    If not, there is either a maximum or minimum at \(c \in (a,b)\text{.}\) Thus \(f'(c) = 0\) by the Interior Extremum Theorem.
    ðŸ”—
    ðŸ”—
    We will prove the MVT by reducing it to Rolleâ€™s Theorem.
    ðŸ”—

    Proof.

    Let
    \begin{equation*} d(x) = f(x)- \left[ \left( \frac{f(b)-f(a)}{b-a} \right)(x-a)+ f(a) \right] \end{equation*}
    ðŸ”—
    The term in the bracket is the equation of the line between \((a, f(a))\) and \((b, f(b))\text{.}\)
    ðŸ”—
    d(x) is continuous on \([a,b]\text{,}\) differentiable on \((a,b)\) and \(d(a) = d(b) = 0\text{.}\) Thus \(d(x)\) satisfies the hypotheses of Rolleâ€™s Theorem.
    ðŸ”—
    it follows that
    \begin{equation*} 0 = d'(c) = f'(c)- \frac{f(b)-f(a)}{b-a} \end{equation*}
    ðŸ”—
    Hence \(f'(c) = \frac{f(b)-f(a)}{b-a}\text{.}\)
    ðŸ”—
    ðŸ”—
    As an illustration of the power of the MVT we will prove the following results about derivatives.
    ðŸ”—

    Corollary 18.

    If \(g: A \to \mathbf{R}\) is differentiable on an interval \(A\) and satisfies \(g'(x) = 0\) for all \(x \in A\text{,}\) then \(g(x)=k\) for some constant \(k \in \mathbf{R}\text{.}\)
    ðŸ”—
    ðŸ”—

    Corollary 19.

    If \(f\) and \(g\) are differentiable on an interval \(A\) and satisfy \(f'(x) = g'(x)\) for all \(x \in A\) then \(f(x) = g(x) +k\) for some constant \(k \in \mathbf{R}\)
    ðŸ”—
    ðŸ”—

    Subsection Increasing and Decreasing

    Recall that a function is increasing on an interval \((a,b)\) if \(f(x)\leq f(y)\) whenever \(x < y\) in \((a,b)\text{.}\) Likewise a function is decreasing on an interval \((a,b)\) if \(f(x)\geq f(y)\) whenever \(x < y\) in \((a,b)\text{.}\)
    ðŸ”—
    It is taught in Calculus class that a function is increasing whenever the derivative is positive and decreasing whenever the derivative is negative, but this is not quite right.
    ðŸ”—
    Consider the function
    \begin{equation*} g(x) = \begin{cases} \hfill \frac{x}{2}+ x^2 \sin(1/x) \amp \text{ if $x \neq 0$} \\ \hfill 0 \hfill \amp \text{ if $x =0 $} \\ \end{cases} \end{equation*}
    ðŸ”—
    Let \(y = \frac{x}{2}+ x^2 \sin(1/x)\text{.}\) Then \(g'(0) = \displaystyle \lim_{x\to 0} \frac{y}{x} = \frac{1}{2}>0\text{.}\)
    ðŸ”—
    Let \(k\) be a positive integer. Then \(g(\frac{1}{2\pi k}) = \frac{1}{2\cdot 2\pi k}\) but \(g(\frac{1}{2 \pi k + \frac{\pi}{2}}) = \frac{1}{2\cdot 2\pi k + \pi}+ \frac{1}{(2 \pi k + \frac{\pi}{2})^2}\)
    ðŸ”—
    \(\frac{1}{2\cdot 2\pi k}-\frac{1}{2\cdot 2\pi k + \pi} = \frac{\pi}{(4\pi k)^2 }\) but \(\frac{1}{(2 \pi k + \frac{\pi}{2})^2}>\frac{1}{(3\pi k)^2 }\)
    ðŸ”—
    When \(k\) is large \(\frac{1}{(3\pi k)^2 } > \frac{\pi}{(4\pi k)^2 }\) and hence \(g(\frac{1}{2\pi k}) < g(\frac{1}{2 \pi k + \frac{\pi}{2}})\) but \(\frac{1}{2 \pi k + \frac{\pi}{2}} < \frac{1}{2\pi k} \text{.}\) Therefore \(f\) is non-increasing on every neighborhood of \(0\text{.}\)
    ðŸ”—

    Theorem 21.

    Suppose \(f: (a,b) \to \mathbf{R}\text{.}\)
    1. If \(f'(x) > 0\) on \((a,b)\) then \(f(x)\) is increasing on \((a,b)\text{.}\)
      ðŸ”—
      ðŸ”—
    2. If \(f'(x) < 0\) on \((a,b)\) then \(f(x)\) is decreasing on \((a,b)\text{.}\)
      ðŸ”—
      ðŸ”—

    Subsection The Generalized Mean Value Theorem

    There is a more general statement of the MVT, whose proof is also due to Cauchy. For this reason it is often referred to as Cauchyâ€™s Mean Value Theorem.
    ðŸ”—

    Theorem 22. Generalized Mean Value Theorem.

    If \(f\) and \(g\) are continuous on the closed interval \([a,b]\) and differentiable on the open interval \((a,b)\) then there exists a point \(c\in(a,b)\) where
    \begin{equation*} [f(b)-f(a)]g'(c) = [g(b)-g(a)]f'(c). \end{equation*}
    ðŸ”—
    ðŸ”—
    If \(g'\) is never zero on \((a,b)\) then the conclusion can be stated as
    \begin{equation*} \frac{f'(c)}{g'(c)} = \frac{f(b)-f(a)}{g(b)-g(a)} \end{equation*}
    ðŸ”—

    Proof.

    We will apply the MVT to the function \(h(x) = [f(b)-f(a)]g(x)-[g(b)-g(a)]f(x)\text{.}\)

    Subsection Intermediate Forms and Lâ€™Hopitalâ€™s Rule

    Consider the following limits.
    \begin{equation*} \lim_{x\to 1} \frac{\ln(x)}{x-1} \end{equation*}
    \begin{equation*} \lim_{x\to 0^+} \frac{\ln(x)}{\csc(x)} \end{equation*}
    ðŸ”—
    If we try to evaluate the first limit we end up with both numerator and denominator going to \(0\text{.}\) This is called an indeterminate form of type \(\frac{0}{0}\)
    ðŸ”—
    If we try to evaluate the second limit we end up with the numerator going to \(-\infty\) and the denominator going to \(\infty\text{.}\) This is called an indeterminate form of type \(\frac{\infty}{\infty}\)
    ðŸ”—
    Before proceeding, letâ€™s formally define what is meant for a limit to be \(\infty\text{.}\)
    ðŸ”—

    Definition 23.

    Given \(g:A \to \mathbf{R}\) and a limit point \(c\) of \(A\text{,}\) we say that \(\displaystyle \lim_{x\to c} g(x) = \infty\) if for every \(M > 0\) there exists \(\delta > 0\) such that whenever \(0 < |x-c| < \delta\) it follows that \(g(x) \geq M\text{.}\)
    ðŸ”—
    ðŸ”—
    The definition for \(\displaystyle \lim_{x\to c} g(x) = -\infty\) is similar.
    ðŸ”—

    Theorem 24. Lâ€™hopitalâ€™s Rule (\(\frac{0}{0}\) case).

    Let \(f\) and \(g\) be continuous on an interval containing \(a\) and suppose \(f\) and \(g\) are differentiable on this interval with the possible exception of the point \(a\text{.}\) If \(f(a) = g(a) = 0\) and \(g'(x) \neq 0\) for all \(x \neq a\) then
    \begin{equation*} \lim_{x\to a} \frac{f'(x)}{g'(x)} = L \textnormal{ implies } \lim_{x\to a} \frac{f(x)}{g(x)} = L \end{equation*}
    ðŸ”—
    ðŸ”—

    Theorem 25. Lâ€™hopitalâ€™s Rule (\(\frac{\infty}{\infty}\) case).

    Let \(f\) and \(g\) be differentiable on an interval containing \(a\) with the possible exception of the point \(a\text{.}\) If \(\displaystyle \lim_{x\to a} = \infty\) (or \(-\infty)\) and \(g'(x) \neq 0\) for all \(x \neq a\) then
    \begin{equation*} \lim_{x\to a} \frac{f'(x)}{g'(x)} = L \textnormal{ implies } \lim_{x\to a} \frac{f(x)}{g(x)} = L \end{equation*}
    ðŸ”—
    ðŸ”—
    Lâ€™Hospitalâ€™s rule works because if \(f\) and \(g\) are two differentiable functions whose graph approaches \(0\) as \(x \to a\) then the tangent lines to the graphs at \(a\) are close to the functions themselves, so we can replace the functions by their tangent lines at \(a\text{.}\)
    ðŸ”—
    Recall that \(f'(a)\) is the slope of the tangent line to \(f\) and \(g'(a)\) is the slope of the tangent line to \(g\text{.}\) Thus
    \begin{equation*} \lim_{x\to a} \frac{f(x)}{g(x)} = \lim_{x\to a} \frac{f'(x) (x-a) }{g'(x) (x-a)} = \lim_{x\to a} \frac{f'(x) }{g'(x)} \end{equation*}
    ðŸ”—
    The first proof of Lâ€™Hospitalsâ€™s rule appeared in Lâ€™Hospitalâ€™s 1696 book \textit{Infinitesimal Analysis to Understand Curved Lines}, which was the first ever calculus textbook; however, the proof of Lâ€™Hospitalâ€™s rule was due to Johann Bernoulli.
    ðŸ”—
    Lâ€™Hospital had earlier made the following proposition to Bernoulli. I will be happy to give you a retainer of 300 pounds, beginning with the first of January of this year. ... I promise shortly to increase this retainer, which I know is very modest, as soon as my affairs are somewhat straightened out. ... I am not so unreasonable as to demand in return all of your time, but I will ask you to give me at intervals some hours of your time to work on what I request and also to communicate to me your discoveries, at the same time asking you not to disclose any of them to others.
    ðŸ”—
    We will first prove the \(\frac{0}{0}\) case using the Generalized MVT
    ðŸ”—

    Proof.

    Consider the interval \((a,x)\text{.}\) By the Generalized MVT for each \(x\) there exists \(x^* \in (a,x)\) such that
    \begin{equation*} \frac{f'(x^*)}{g'(x^*)} = \frac{f(x)-f(a)}{g(x)-g(a)} = \frac{f(x)}{g(x)} \end{equation*}
    ðŸ”—
    Taking the limit as \(x \to a^+\) yields
    \begin{equation*} \lim_{x\to a^+} \frac{f'(x)}{g'(x)} = \lim_{x\to a^+} \frac{f(x)}{g(x)} \end{equation*}
    ðŸ”—
    The limit as \(x \to a^-\) is similar.
    ðŸ”—
    ðŸ”—
    Now we will prove the \(\frac{\infty}{\infty}\) case of Lâ€™Hospitalâ€™s rule
    ðŸ”—

    Proof.

    Let \(\epsilon>0\text{.}\) Then since \(\displaystyle \lim_{x\to a} \frac{f'(x)}{g'(x)} = L\text{,}\) there exists \(\delta_1>0\) such that
    \begin{equation*} \left| \frac{f'(x)}{g'(x)}-L \right| < \frac{\epsilon}{2} \end{equation*}
    for all \(a < x < a+\delta_1\text{.}\)
    ðŸ”—
    Let \(t = a+\delta_1\text{.}\) Note that \(t\) is a fixed number throughout the remainder of the proof.
    ðŸ”—
    For any \(x \in (a,t)\) the Generalized MVT yields
    \begin{equation*} \frac{f(x)-f(t)}{g(x)-g(t)} = \frac{f'(c)}{g'(c)} \end{equation*}
    for some \(c \in (x,t)\text{.}\)
    ðŸ”—
    It thus follows from the choice of \(t\) that
    \begin{equation} L- \frac{\epsilon}{2} < \frac{f(x)-f(t)}{g(x)-g(t)} < L+ \frac{\epsilon}{2}\tag{0} \end{equation}
    ðŸ”—
    Our goal is to show that
    \begin{equation*} L- \epsilon < \frac{f(x)}{g(x)} < L + \epsilon \end{equation*}
    ðŸ”—
    To achieve this from what we have, we want to multiply (0) by \((g(x)-g(t))/g(x)\text{.}\) To make sure that this quantity is positive note that since \(\displaystyle \lim_{x\to a} g(x) = \infty\text{,}\) we can choose \(\delta_2\) so that \(g(x) > g(t)\) for all \(a < x < a +\delta_2\text{.}\)
    ðŸ”—
    Performing the desired multiplication results in
    \begin{equation*} \left(L - \frac{\epsilon}{2} \right)\left(1-\frac{g(t)}{g(x)}\right) < \frac{f(x)-f(t)}{g(x)} < \left(L + \frac{\epsilon}{2} \right)\left(1-\frac{g(t)}{g(x)}\right) \end{equation*}
    ðŸ”—
    After some algebra this reduces to
    \begin{equation*} L- \frac{\epsilon}{2} + \frac{-Lg(t)+\frac{\epsilon}{2}g(t)+f(t)}{g(x)} < \frac{f(x)}{g(x)} < L+ \frac{\epsilon}{2} + \frac{-Lg(t)-\frac{\epsilon}{2}g(t)+f(t)}{g(x)} \end{equation*}
    ðŸ”—
    Since \(t\) is fixed and \(\displaystyle \lim_{x\to a} g(x) = \infty\) there exists \(\delta_3 > 0\) such that for \(a < x < a+\delta_3\text{,}\) \(\frac{-Lg(t)+\frac{\epsilon}{2}g(t)+f(t)}{g(x)}\) and \(\frac{-Lg(t)-\frac{\epsilon}{2}g(t)+f(t)}{g(x)}\) are both less than \(\frac{\epsilon}{2}\) in absolute value.
    ðŸ”—
    If \(\delta = \min \{\delta_1, \delta_2, \delta_3 \}\) then
    \begin{equation*} \left|\frac{f(x)}{g(x)} -L \right| < \epsilon \end{equation*}
    for all \(a < x < a+\delta\text{.}\)
