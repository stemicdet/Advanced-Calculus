    Section Week 3: A Continuous Nowhere Differentiable Function (5.4)

    Subsection Non-differentiability

    We saw that the absolute value function fails to be differentiable at \(0\) because of the sharp corner at \(0\text{.}\)
    ðŸ”—
    Given a finite set of points, it is possible to construct a continuous function with a sharp corner at each point and thus we have a continuous function that fails to be differentiable at a finite set of points.
    ðŸ”—
    It is possible to construct a continuous function that is nondifferentiable at infinitely many points. Let \(h(x) = |x|\) on the interval \([-1,1]\) and then extend \(h(x)\) to all \(\mathbf{R}\) through the relation \(h(x+2) = h(x)\text{.}\) This function is nondifferentiable at all the integers.
    ðŸ”—
    Is it possible to construct a function that is continuous everywhere but that is non-differentiable at uncountably many points?
    ðŸ”—
    Around 1830 Bernard Bolzano constructed a function that is continuous everywhere but fails to be differentiable at every point in \(\mathbf{R}\text{;}\) however, he never published this result.
    ðŸ”—
    About 1860 Charles Cellerier constructed another function that is everywhere continuous but nowhere differentiable. He also never published his result.
    ðŸ”—
    The first published example of a function that is everywhere continuous and nowhere differentiable is due to Karl Weierstrass in 1872.
    ðŸ”—
    Quotes about the Weierstrass Function:
    • An outrage against common sense -- Henri Poincare
      ðŸ”—
      ðŸ”—
    • I turn with terror and horror from this lamentable scourge of functions with no derivatives. -- Charles Hermite
      ðŸ”—
      ðŸ”—
    • [Up to that time] one would have said that a continuous function is essentially capable of being represented by a curve, and that a curve has always a tangent. Such reasoning has no mathematical value whatever; it is founded on intuition, or rather on a visible representation. ... By the discovery of this striking example Weierstrass has accordingly given us a useful reminder, and has taught us better to appreciate the faultless and purely arithmetical methods with which he more than any one has enriched our science. -- Henri Poincare
      ðŸ”—
      ðŸ”—
    ðŸ”—
    Although nowhere differentiable continuous functions were created to challenge previously held beliefs in mathematics, they do have important physical applications.
    ðŸ”—
    In 1905 Albert Einstein published a paper which demonstrated the existence of atoms. In this paper Einstein showed that atoms exhibit Brownian Motion which means that they follow a random path because they are constantly bombarded by other particles which changes their direction of motion.
    ðŸ”—
    Because of this randomness, the path of an object that exhibits Brownian motion canâ€™t be modeled by a smooth curve. One must use a continuous non-differentiable function to model Brownian motion.
    ðŸ”—
    Weierstrassâ€™ original function was the function
    \begin{equation*} f(x) = \sum_{n=0}^{\infty} a_n \cos(b^n x) \end{equation*}
    where \(a\) and \(b\) are carefully chosen real numbers.
    ðŸ”—
    The Weierstrass function is an example of a Fourier series, which is an infinite sum of sines and cosines.
    ðŸ”—
    Remarkably the Weierstrass function is nowhere differentiable despite the fact that the functions in the sum are infinitely differentiable everywhere.
    ðŸ”—
    We will give another example of an everywhere continuous nowhere differentiable function. Recall that \(h(x) = |x|\) on \([-1,1]\) and is extended to all \(\mathbf{R}\) by the functional equation \(h(x+2) = h(x)\text{.}\)
    ðŸ”—
    Let \(h_n(x) = \frac{1}{2^n}h(2^nx)\text{.}\)
    ðŸ”—
    Let
    \begin{equation*} g(x) = \sum_{n=0}^{\infty} h_n(x) = \sum_{n=0}^{\infty}\frac{1}{2^n}h(2^nx) \end{equation*}
    ðŸ”—
    We will show that \(g(x)\) is continuous on all \(\mathbf{R}\) but differentiable nowhere.

    Subsubsection Continuity of \(g(x)\)

    We first need to show that \(g(x)\) is well defined for all \(x \in \mathbf{R}\) by showing that the infinite sum converges for all \(x\text{.}\)
    ðŸ”—
    Since \(|h(x)|\leq 1\text{,}\) \(\frac{1}{2^n}h(2^n x) \leq \frac{1}{2^n}\text{.}\) Hence by the comparison test
    \begin{equation*} g(x)= \sum_{n=0}^{\infty}\frac{1}{2^n}h(2^nx) \leq \sum_{n=0}^{\infty}\frac{1}{2^n} = 2 \end{equation*}
    Thus \(g(x)\) is defined for all \(x \in \mathbf{R}\text{.}\)
    ðŸ”—
    Since \(h(x)\) is continuous it follows from the algebraic limit theorem that the finite sum
    \begin{equation*} g_m(x) = \sum_{n=0}^m \frac{1}{2^n}h(2^n x) \end{equation*}
    is continuous.
    ðŸ”—
    It is not necessarily the case that an infinite sum of continuous functions will still be continuous. Determining when the limit of a sequence of continuous functions or an infinite sum of functions is continuous will be one of the main goals of Chapter 6.
    ðŸ”—
    After that chapter we will see that this particular sum is in fact continuous on all \(\mathbf{R}\)
    ðŸ”—
    ðŸ”—

    Subsubsection Non-differentiability of \(g(x)\)

    We will first show that \(g(x)\) is not differentiable at \(0\text{.}\) Consider the sequence \(x_m = 1/2^m\text{,}\) where \(m = 0, 1, 2, \ldots\)
    ðŸ”—
    It follows from the definition of the derivative that if \(g(x)\) is differentiable at \(0\) then
    \begin{equation*} g'(0) = \lim_{m\to \infty} \frac{g(x_m)-g(0)}{x_m - 0} = \lim_{m\to \infty} \frac{\displaystyle \sum_{n=0}^{\infty} 2^{-n}h(2^{n-m})}{2^{-m}} \end{equation*}
    ðŸ”—
    There are three cases to consider:
    1. \(m < n\text{:}\) Then \(2^{n-m}\) is a multiple of 2 and thus \(h(2^{m-n})=0\)
      ðŸ”—
      ðŸ”—
    2. Case 2: \(m=n\text{:}\) Then \(2^{n-m} = 1\) so \(h(2^{m-n}) = 1\text{.}\)
      ðŸ”—
      ðŸ”—
    3. Case 3: \(m>n\text{:}\) Then \(2^{n-m} < 1\) and hence \(h(2^{m-n}) = 2^{m-n}\text{.}\)
      ðŸ”—
      ðŸ”—
    ðŸ”—
    Putting this all together we get
    \begin{align*} g'(0) \amp = \lim_{m\to \infty} \frac{\displaystyle \sum_{n=0}^{m-1} 2^{-n}h(2^{n-m}) + 2^{-m} + \sum_{n=m+1}^{\infty} 2^{-n}h(2^{n-m})}{2^{-m}}\\ \amp =\lim_{m\to \infty} 2^m \left(\sum_{n=0}^{m-1} 2^{-m} + 2^{-m} \right)\\ \amp =\lim_{m\to \infty} m+1 \end{align*}
    ðŸ”—
    Since the limit is undefined, \(g'(0)\) does not exist and \(g\) is not differentiable at \(0\text{.}\)
    ðŸ”—
    A similar argument shows that \(g'(x)\) does not exist whenever \(x\) is a rational number of the form \(\frac{p}{2^k}\) where \(p\) is an integer and \(k\) is a positive integer.
    ðŸ”—
    These points are called dyadic points. Because the dyadic points are dense in the real numbers there is reason to suppose that \(g(x)\) is non-differentiable at every real number.
    ðŸ”—
    Suppose that \(x\) is not a dyadic number. Then for a fixed value of \(m\) \(x\) lies between 2 adjacent dyadic points
    \begin{equation*} \frac{p_m}{2^m} < x < \frac{p_m+1}{2^m} \end{equation*}
    ðŸ”—
    Set \(x_m = \frac{p_m}{2^m}\) and \(y_m = \frac{p_m+1}{2^m}\text{.}\) Repeating this for each \(m\) yields sequences \((x_m)\) and \((y_m)\) such that \(\lim x_m = \lim y_m = x\) and \(x_m < x < y_m\)
    ðŸ”—

    Lemma 30.

    If \((a_n)\) and \((b_n)\) are sequences satisfying \(a_n < a < b_n\) and \(\lim a_n = \lim b_n = a\) then
    \begin{equation*} f'(a) = \lim_{n \to \infty} \frac{f(b_n)-f(a_n)}{b_n - a_n}. \end{equation*}
    ðŸ”—
    ðŸ”—

    Proof.

    \begin{equation*} f'(a) = \lim_{x\to a} \frac{f(x)-f(a)}{x - a} \end{equation*}
    ðŸ”—
    Let \(x = b_n\text{.}\) Then \(x\to a\) as \(n \to \infty\) so
    \begin{equation*} f'(a) = \lim_{n\to \infty} \frac{f(b_n)-f(a)}{b_n - a} \end{equation*}
    But \(a = \displaystyle \lim_{n\to \infty} a_n \text{.}\) Thus
    \begin{equation*} f'(a) = \lim_{n \to \infty} \frac{f(b_n)-f(a_n)}{b_n - a_n}. \end{equation*}
    ðŸ”—
    ðŸ”—
    We will use the lemma to show that \(g'(x)\) canâ€™t exist for all \(x\text{.}\) Recall that \(x_m = \frac{p_m}{2^m}\) and \(y_m = \frac{p_m+1}{2^m}\) are sequences such that \(\lim x_m = \lim y_m = x\) and \(x_m < x < y_m\text{.}\)
    ðŸ”—
    Thus if \(g'(x)\) exists then by the lemma
    \begin{equation*} g'(x) = \lim_{m \to \infty} \frac{g(y_m)-g(x_m)}{y_m-x_m} = \lim_{m \to \infty} \frac{g(\frac{p_m+1}{2^m})-g(\frac{p_m}{2^m})}{2^{-m}} \end{equation*}
    ðŸ”—
    Now
    \begin{equation*} g(\frac{p_m}{2^m}) = \sum_{n=0}^{m-1} 2^{-n}h(\frac{2^n p_m}{2^m}) + 2^{-m}h(p_m) + \sum_{n=m+1}^{\infty} 2^{-n}h(\frac{2^n p_m}{2^m}) \end{equation*}
    ðŸ”—
    If \(n > m\text{,}\) then \(h(\frac{2^n p_m}{2^m}) = h(2^{n-m}p_m)= 0\text{.}\)
    ðŸ”—
    If \(n <m\) then \(h(\frac{2^n p_m}{2^m}) = \pm 1\text{,}\) where the sign depends on whether \(\frac{2^n p_m}{2^m}\) lies on a line segment with positive slope or negative slope.
    ðŸ”—
    Finally \(h(p_m)\) = \(1\) if \(p_m\) is odd and \(0\) if \(p_m\) is even.
    ðŸ”—
    For \(g(\frac{p_m+1}{2^m})\) everything is the same except that \(h(p_m+1) = 0\) if \(p_m\) is odd and \(1\) if \(p_m\) is even.
    ðŸ”—
    Thus \(g(\frac{p_m+1}{2^m})-g(\frac{p_m}{2^m}) = 2^{-m}\) if \(p_m\) is even and \(-2^{-m}\) if \(p_m\) is odd.
    ðŸ”—
    Therefore
    \begin{equation*} g'(x) = \lim_{m \to \infty} \frac{g(\frac{p_m+1}{2^m})-g(\frac{p_m}{2^m})}{2^{-m}} = \lim_{m \to \infty} \frac{\pm 2^{-m}}{2^{-m}} \end{equation*}
    where the sign is positive if \(p_m\) is even and negative if \(p_m\) is odd.
    ðŸ”—
    Since this sequence alternates between \(-1\) and \(1\text{,}\) the limit doesnâ€™t exist and thus \(g\) is not differentiable at \(x\text{.}\) Since \(x\) was arbitrary, this shows that \(g\) is nowhere differentiable.
