    Section Week 4: Uniform Convergence of a sequence of functions (6.1-6.2)

    Subsection Function Approximation

    Given a relatively complicated function we might wish to approximate it by simpler functions such as polynomials, rational functions, trigonometric functions or exponential functions. This way we could simplify calculations involving the more complicated function.
    ðŸ”—
    There are several noteworthy results which show how a large number of functions can be approximated by relatively simple functions.
    1. Taylor series: Given a function \(f\) that is infinitely differentiable at a point \(x_0\) there is a sequence of polynomials that approximates \(f\) near \(x_0\text{.}\)
      ðŸ”—
      ðŸ”—
    2. Fourier series: Similar to a Taylor series but with sines and cosines (or complex exponentials) used as approximants instead of polynomials.
      ðŸ”—
      ðŸ”—
    3. Rungeâ€™s Theorem (1885): If \(K\) is a compact subset of the complex plane then every function that is analytic (i.e. complex differentiable) in a neighborhood of \(K\) can be uniformly approximated by a sequence of rational functions.
      ðŸ”—
      ðŸ”—
    4. Weierstrass Approximation Theorem (1885): Every continuous function on \(\mathbf{R}\) can be uniformly approximated by polynomials.
      ðŸ”—
      ðŸ”—
    5. Mergelyanâ€™s Theorem (1951): Let \(K\) be a compact subset of the complex plane such that \(K^c\) is connected. Then every function analytic on a neighborhood of \(K\) can be uniformly approximated by polynomials.
      ðŸ”—
      ðŸ”—
    ðŸ”—
    A function \(f\) can be approximated by polynomials (rational functions) if there is a sequence \((f_n)\) of polynomials (rational functions) that converges to \(f\text{.}\)
    ðŸ”—
    Given one of these approximations, we might ask which properties of the approximating functions extend to the limit functions. For example
    1. If \(f(x)\) is the limit of a Fourier series, is \(f(x)\) a continuous function?
      ðŸ”—
      ðŸ”—
    2. If \(g(x)\) is the limit of a sequence of polynomials, is \(g(x)\) necessarily a differentiable?
      ðŸ”—
      ðŸ”—
    3. In this chapter, we will investigate these and other questions concerning approximation.
      ðŸ”—
      ðŸ”—

    Subsection Pointwise Convergence

    The first thing we need to do to build up a theory of function approximation is to define what it means for a sequence of functions to converge.
    ðŸ”—
    As it turns out, there are multiple different ways that we can define what it means for a sequence of functions to converge. We will start by extending the notion of converge of a sequence from last semester.
    ðŸ”—

    Definition 31.

    For each \(n \in \mathbf{N}\) let \(f_n\) be a function defined on a set \(A \subseteq \mathbf{R}\text{.}\) The sequence \((f_n)\) converges pointwise on \(A\) to a function \(f\) if for all \(x \in A\text{,}\) the sequence of real numbers \(f_n(x)\) converges to \(f(x)\text{.}\)
    ðŸ”—
    ðŸ”—
    In this case we write \(f_n \to f\text{,}\) \(\lim f_n = f\text{,}\) or \(\displaystyle \lim_{n\to \infty} f_n(x) = f(x)\)
    ðŸ”—
    The questions that weâ€™ve raised about whether the limit of a sequence of continuous (or differentiable) functions is itself continuous (or differentiable) is really a question about when we are allowed to interchange limits.
    ðŸ”—
    Suppose \(f_n \to f\) and each \(f_n\) is continuous at \(c\text{.}\) For \(f\) to be continuous at \(c\text{,}\) we must show that
    \begin{equation*} \lim_{x\to c} f(x) = f(c). \end{equation*}
    ðŸ”—
    Note that
    \begin{equation*} \lim_{x\to c} f(x) = \lim_{x\to c} \lim_{n \to \infty} f_n(x) = \lim_{n \to \infty} \lim_{x\to c} f_n(x) = \lim_{n \to \infty} f_n(c) = f(c) \end{equation*}
    but only if the interchange of limits is justified.
    ðŸ”—
    ðŸ”—

    Subsection Uniform Convergence

    The second example shows that if a sequence of continuous functions converges pointwise to a limit function, the limit function is not necessarily a continuous function.
    ðŸ”—
    Cauchy 1821: Theorem that if a sum of continuous functions converges then the limit function is continuous.
    ðŸ”—
    Abel 1826: Cauchyâ€™s theorem admits exceptions such as \(\sin \phi -\frac{1}{2} \sin 2\phi + \frac{1}{3} \sin 3\phi - \ldots\) , which is discontinuous at every value of \((2m+1)\pi\) , where \(m\) is an integer.
    ðŸ”—
    It has been argued; however, that Cauchyâ€™s theorem is not wrong and that Cauchy knew about Abelâ€™s exception but it is not a counterexample to his theorem.
    ðŸ”—
    The issue involves what did Cauchy mean by a convergent sum. Abel interpreted Cauchyâ€™s theorem as pointwise convergence, which is false as his exception shows.
    ðŸ”—
    However, Cauchy may have meant for convergence to mean a stronger type than pointwise convergence, which today is called uniform convergence.
    ðŸ”—
    Abelâ€™s exception is not uniformly convergent and thus would not be a counterexample to Cauchyâ€™s theorem. In fact, Cauchyâ€™s theorem is correct if the convergence is uniform convergence.
    ðŸ”—
    Before introducing the definition of uniform convergence, letâ€™s restate the definition of pointwise convergence in terms of \(\epsilon\) .
    ðŸ”—

    Definition 32.

    Let \((f_n)\) be a sequence of functions defined on a set \(A \subseteq \mathbf{R}\) . Then \((f_n)\) converges pointwise on \(A\) to a limit \(f\) defined on \(A\) if, for every \(\epsilon > 0\) and \(x \in A\) , there exists \(N(\epsilon,x) \in \mathbf{N}\) (perhaps dependent on \(x\) ) such that \(|f_n(x)-f(x)| < \epsilon\) whenever \(n \geq N\) .
    ðŸ”—
    ðŸ”—
    Notice that the choice of \(N\) may depend on the point \(x\) .
    ðŸ”—

    Definition 33.

    Let \((f_n)\) be a sequence of functions on a set \(A \subseteq \mathbf{R}\) . Then \((f_n)\) converges uniformly on \(A\) to a limit function defined on \(A\) if for every \(\epsilon > 0\) there exists \(N(\epsilon) \in \mathbf{N}\) such that \(|f_n(x)-f(x)|< \epsilon\) whenever \(n \geq N\) and \(x \in A\) .
    ðŸ”—
    ðŸ”—
    The difference between uniform and convergence and pointwise convergence is that in uniform convergence for each \(\epsilon>0\) a single \(N\) must work for all \(x \in A\) while in pointwise convergence for each \(\epsilon >0\) and \(x \in A\) different choices of \(N\) may be used.
    ðŸ”—
    Let \(f_n(x) = \frac{\sin(x)}{n}\) . Show that \(f_n(x)\) converges uniformly to \(0\) on \(\mathbf{R}\) .
    ðŸ”—

    We saw that the sequence \(g_n(x) = x^n\) converges pointwise on \([0,1]\) to
    \begin{equation*} g(x) = \begin{cases} \hfill 0 \amp \text{ if $ 0 \leq x < 1$ } \\ \hfill 1 \hfill \amp \text{ if $x = 1$ } \end{cases} \end{equation*}
    ðŸ”—
    Letâ€™s show that this sequence does not converge uniformly on \([0,1]\) . For this, suppose \(0 < x < 1\) and \(0 < \epsilon < 1\) .
    ðŸ”—
    If the convergence was uniform then we would be able to find \(N(\epsilon) \in \mathbf{N}\) depending only on \(\epsilon\) such that \(|g_n(x)-g(x)| < \epsilon\) when \(n \geq N\) . But
    \begin{equation*} |g_n(x)-g(x)| = |x^n-0| = x^n \end{equation*}
    ðŸ”—
    If \(x^n < \epsilon\) then
    \begin{align*} \ln(x^n) < \amp \ln(\epsilon)\\ n \ln(x) < \amp \ln(\epsilon) \\ n > \amp \frac{\ln(\epsilon)}{\ln(x)} \end{align*}
    ðŸ”—
    But this means that \(n\) depends on the choice of \(x\) (note that as \(x\to 1^-\) , \(n \to \infty\) ) and thus the convergence is not uniform.
    ðŸ”—

    Subsection Uniform Convergence and Continuity

    In the two examples of sequences of continuous functions that converge to a limit function that is not continuous, the convergence has been pointwise convergence but not uniform convergence. In fact, if a sequence of continuous functions converges uniformly, then the limit function will also be continuous.
    ðŸ”—

    Theorem 34. Continuous Limit Theorem.

    Let \((f_n)\) be a sequence of functions defined on \(A \subseteq \mathbf{R}\) that converges uniformly on \(A\) to a function \(f\text{.}\) If each \(f_n\) is continuous at \(c \in A\text{,}\) then \(f\) is continuous at \(c\text{.}\)
    ðŸ”—
    ðŸ”—

    Proof.

    Let \(c \in A\) be fixed and let \(\epsilon >0\text{.}\) It follows from the definition of uniform continuity that there exists \(N(\epsilon)\in \mathbf{N}\) such that \(|f_n(x)-f(x)| < \epsilon\) whenever \(n \geq N\text{.}\)
    ðŸ”—
    Because \(f_N\) is continuous at \(c\text{,}\) there exists \(\delta > 0\) such that \(|f_N(x)-f_N(c)| < \epsilon\) whenever \(|x-c| <\delta\text{.}\)
    ðŸ”—
    We want to show that \(|f(x)-f(c)|\) can be made arbitrarily small by choosing \(|x-c|\) sufficiently small.
    ðŸ”—
    Putting this all together we get that if \(|x-c| < \delta\)
    \begin{align*} |f(x)-f(c)| \amp = |f(x)-f_N(x) + f_N(x) -f_N(c)+f_N(c)-f(c)|\\ \amp \leq |f(x)-f_N(x)| + |f_N(x)-f_N(c)| + |f_N(c)-f(c)| \\ \amp < \epsilon + \epsilon + \epsilon = 3\epsilon \end{align*}
    ðŸ”—
    Since \(\epsilon\) can be made arbitrarily small, so can \(3\epsilon\) and thus \(f(x)\) is continuous at \(c\text{.}\)
    ðŸ”—
    ðŸ”—
    Something to consider: If convergence is only pointwise, where does the proof break down?
    ðŸ”—
    ðŸ”—

    Subsection Cauchy Criterion

    There is a Cauchy criterion for uniform convergence similar to the one for sequences.

    Theorem 35.

    A sequence of functions \((f_n)\) converges uniformly on \(A\) if and only if for every \(\epsilon>0\) there exists \(N(\epsilon) \in \mathbf{N}\) such that \(|f_n(x)-f_m(x)| < \epsilon\) whenever \(m,n \geq N\) and \(x \in A\text{.}\)
