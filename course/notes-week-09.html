    Section Week 9: The Reimann Integral (7.1-7.2)

    Subsection History of the Integral

    The ancient Greek mathematicians found the areas of geometric figures using the ``method of exhaustion’’. They found the areas of polygons by dividing them into triangles and adding the areas of the triangles together.
    🔗
    The area of a circle was approximated by dividing the circle into a large number of triangles, finding the areas of the triangles and adding them together.
    🔗
    The integral of a function as opposed to the area of a geometric figure was first considered by the early founders of calculus such as Newton and Leibniz. They viewed the integral as the inverse process of differentiation.
    🔗

    Definition 57.

    Newton-Leibniz Integral: The integral of the function \(f\) is the function \(F\) such that \(F'(x) = f(x)\text{.}\)
    🔗
    🔗
    This definition has several shortcomings. First it means that the only functions that can be differentiated are derivatives, which is a relatively small class of functions.
    🔗
    Second since derivatives satisfy the Intermediate Value Property this means that any function with a jump discontinuity cannot be integrated.
    🔗
    In 1823 Cauchy developed a more rigorous definition based on the idea of finding areas under a curve. It uses the concept of a partition of an interval.
    🔗

    Definition 58.

    A partition of \([a,b]\) is a finite set of points from \([a,b]\) that includes both \(a\) and \(b\text{.}\) The notational convention is to list the points of a partition \(P = \{x_0, x_1, x_2, \ldots, x_n\}\) in increasing order; thus
    \begin{equation*} a=x_0 < x_1 < x_2 < \ldots < x_n = b \end{equation*}
    🔗
    🔗

    Definition 59.

    Cauchy Integral: Let \(f(x)\) be a continuous function on \([a,b]\) and let \(a = x_0 < x_1 < x_2 < \ldots < x_n = b\text{,}\) and \(\Delta x_k = x_k-x_{k-1}\) for \(k = 1,2, \ldots n\text{.}\) The limit
    \begin{equation*} \lim_{\textnormal{ max } \Delta x_k \to 0} \sum_{k=1}^n f(x_{k-1}) \Delta x_k \end{equation*}
    is called the Cauchy integral of \(f(x)\) over \([a,b]\) and is denoted by
    \begin{equation*} \int_a^b f(x) dx. \end{equation*}
    🔗
    🔗
    Geometrically the Cauchy integral determines the area under the curve \(f(x)\) by dividing \([a,b]\) into rectangles of width \(\Delta x_i\text{.}\) The height of the rectangle is chosen to be the height of \(f(x)\) at the left endpoint of the rectangle.
    🔗
    In taking the limit as the maximum width of the rectangles goes to \(0\) more rectangles are required to cover the area beneath the curve and as the maximum width of the rectangles tends to \(0\) the limit converges to the area under the curve.
    🔗
    Because the Cauchy integral is not defined in terms of antiderivatives it allows for a larger number of functions to be integrated compared to the Newton-Leibniz integral. However, it requires that the function to be integrated be continuous which is still rather restrictive.
    🔗
    In 1868 Bernhard Riemann developed a new way to define integration based off Cauchy’s ideas that would allow for the integration of some discontinuous functions.
    🔗

    Definition 61.

    Riemann Integral: Let \(f(x)\) be a function on \([a,b]\) and let \(a = x_0 < x_1 < x_2 < \ldots < x_n = b\text{,}\) and \(\Delta x_k = x_i-x_{i-1}\) for \(i = 1,2, \ldots n\text{.}\) On each subinterval \([x_{k-1}, x_k]\) choose a point \(c_k \in [x_{k-1}, x_k]\) The limit
    \begin{equation*} \lim_{\textnormal{ max } \Delta x_k \to 0} \sum_{k=1}^n f(c_k) \Delta x_k \end{equation*}
    is called the Riemann integral of \(f(x)\) over \([a,b]\) and is denoted by
    \begin{equation*} \int_a^b f(x) dx. \end{equation*}
    🔗
    🔗
    The sum that appears in the definition of the Riemann integral is called a Riemann sum.
    🔗
    The difference between the Riemann integral and the Cauchy integral is that in the Cauchy integral the height of the rectangle is given by value of \(f(x)\) at the left endpoint, while in the Riemann integral the height of the rectangle is determined by the value of \(f(c_k)\) which can be any point in the subinterval \([x_{k-1}, x_k]\)
    🔗
    The value of the Riemann integral if it exists is independent of the choice of the points \(c_k\text{.}\) No matter what points you choose you will get the same value for the limit of the Riemann sum.
    🔗
    The Riemann integral makes it possible to integrate a much larger set of functions than the Cauchy integral, including functions with a finite number of discontinuities; however, the Riemann integral still has a few drawbacks.
    🔗
    The Riemann integral can’t be used to integrate some functions with uncountably many discontinuities such as Dirichlet’s function
    \begin{equation*} f(x) = \begin{cases} \hfill 1 \amp \text{ if $x \in \mathbf{Q}$} \\ \hfill 0 \hfill \amp \text{ if $x \notin \mathbf{Q}$} \end{cases} \end{equation*}
    🔗
    A sequence of Riemann integrable functions can converge pointwise to a function that is not Riemann integrable.
    🔗
    In 1904 Henri Lebesgue introduced a new definition of the integral which is now known as the Lebesgue integral. With the Lebesgue integral it is possible to integrate functions with uncountably many discontinuities, like Dirichlet’s function, and the limit of a sequence of Lebesgue integrable functions is always a Lebesgue integrable function.
    🔗
    Lebesgue integration is based on a branch of mathematics called measure theory. In order to understand the Lebesgue integral one needs to be familiar with the concepts of measure theory. For more information take MTH 650: Real Variables I.
    🔗
    In 1875 Gaston Darboux developed his version of the Riemann integral which is now called the Darboux integral. The Darboux integral is equivalent to the Riemann integral; that is, a function is Darboux integrable if and only if it is Riemann integrable, and both integrals yield the same value.
    🔗
    The advantage of the Darboux integral over the Riemann integral is that it is simpler to state and does not require the use of a limit.
    🔗
    🔗

    Subsection The Darboux Integral

    The Darboux integral is defined using lower and upper sums.
    🔗
    Recall that a partition \(P\) of \([a,b]\) is a finite set of points that includes \(a\) and \(b\text{.}\) For each subinterval \([x_{k-1},x_k]\) of \(P\text{,}\) let
    \begin{equation*} m_k = \inf\{f(x): x \in [x_{k-1}, x_k] \} \textnormal{ and } M_k = \sup\{f(x): x \in [x_{k-1}, x_k] \}. \end{equation*}
    🔗

    Definition 62.

    The lower sum of \(f\) with respect to \(P\) is given by
    \begin{equation*} L(f,p) = \sum_{k=1}^n m_k(x_k-x_{k-1}). \end{equation*}
    🔗
    🔗

    Definition 63.

    The upper sum of \(f\) with respect to \(P\) is given by
    \begin{equation*} U(f,p) = \sum_{k=1}^n M_k(x_k-x_{k-1}). \end{equation*}
    🔗
    🔗
    Example: Find the upper and lower sums of \(f(x) = 4x-x^2\) on the interval \([0,4]\) with respect to the partition \(P = \{0, 1, 3, 4\}\)
    🔗

    Definition 64.

    A partition \(Q\) is a refinement of a partition \(P\) if \(Q\) contains all the points of \(P\text{;}\) that is, \(P \subseteq Q\text{.}\)
    🔗
    🔗

    Lemma 65.

    If \(P \subseteq Q\) then
    \begin{equation*} L(f,P) \leq L(f,Q) \leq U(f,Q) \leq U(f,P). \end{equation*}
    🔗
    🔗

    Proof.

    The upper sum of \(f\) with respect to \(Q\) is the supremum of a set whose infimum is the lower sum of \(f\) with respect to \(Q\text{,}\) so \(L(f,Q) \leq U(f,Q)\text{.}\)
    🔗
    Suppose \(Q\) is a refinement of \(P\) obtained by adding an additional point \(z\text{.}\) Further suppose \(z \in [x_{K-1},x_K]\) for some \(K\text{.}\)
    🔗
    Then
    \begin{equation*} m_K = \inf\{f(x): x \in [x_{K-1}, x_K] \} \leq \inf\{f(x): x \in [x_{K-1}, z]\} \end{equation*}
    and
    \begin{equation*} m_K = \inf\{f(x): x \in [x_{K-1}, x_K] \} \leq \inf\{f(x): x \in [z, x_K]\} \end{equation*}
    Let
    \begin{equation*} m_K' = \inf\{f(x): x \in [x_{K-1}, z]\} \textnormal{ and } m_K'' = \inf\{f(x): x \in [z, x_K]\} \end{equation*}
    Then \(m_K \leq m_K'\) and \(m_K \leq m_K''\text{.}\)
    🔗
    Hence
    \begin{align*} L(f,P) \amp = \sum_{k=1}^n m_k (x_k - x_{k-1}) = \sum_{k=1}^{K-1} m_k (x_k - x_{k-1}) + m_K(x_K-x_{K-1}) + \sum_{k=K+1}^n m_k (x_k - x_{k-1})\\ \amp = \sum_{k=1}^{K-1} m_k (x_k - x_{k-1}) + m_K(x_K-z)+ m_K(z-x_{K-1}) + \sum_{k=K+1}^n m_k (x_k - x_{k-1})\\ \amp \leq \sum_{k=1}^{K-1} m_k (x_k - x_{k-1}) + m_K'(x_K-z)+ m_K''(z-x_{K-1}) + \sum_{k=K+1}^n m_k (x_k - x_{k-1}) = L(f,Q). \end{align*}
    🔗
    🔗

    Lemma 66.

    If \(P_1\) and \(P_2\) are any two partitions of \([a,b]\) then \(L(f,P_1) \leq U(f, P_2)\)
    🔗
    🔗

    Proof.

    Let \(P = P_1 \cup P_2\text{,}\) which is a refinement of both \(P_1\) and \(P_2\text{.}\) Thus by the previous result,
    \begin{equation*} L(f,P_1) \leq L(f,P) \leq U(f,P) \leq U(f, P_2) \end{equation*}
    🔗
    🔗
    An upper sum always overestimates the exact value of the integral, but by taking a refinement of the partition the upper sum decreases, meaning that it overestimates the integral by a lesser amount.
    🔗
    Likewise a lower sum always underestimates the exact value of the integral, but by taking a refinement of the partition the lower sum decreasing so that it underestimates the integral by a lesser amount.
    🔗
    The idea behind the Darboux integral is that a function is integrable when the upper and lower sums converge to a common value.
    🔗
    Let \(\mathcal{P}\) be the collection of all possible partitions of the interval \([a,b]\text{.}\) The upper integral of \(f\) is defined to be
    \begin{equation*} U(f) = \inf\{U(f,P): P \in \mathcal{P}\} \end{equation*}
    🔗
    Similarly the lower integral of \(f\) on \([a,b]\) is defined to be
    \begin{equation*} L(f) = \sup\{L(f,P): P \in \mathcal{P}\} \end{equation*}
    🔗

    Lemma 67.

    For any bounded function \(f\) on \([a,b]\text{,}\) it is always the case that \(U(f) \geq L(f)\text{.}\)
    🔗
    🔗

    Definition 68.

    A bounded function \(f\) defined on the interval \([a,b]\) is Darboux Integrable if \(U(f) = L(f)\text{.}\) In this case we define \(\int_a^b f(x)dx\) to be this common value; that is,
    \begin{equation*} \int_a^b f(x) dx = U(f) = L(f) \end{equation*}
    🔗
    🔗
    Note: The textbook uses the term Riemann integrability instead of Darboux integrability. This is not wrong because the Riemann integral and the Darboux integral are equivalent, but it is more precise to use the term Darboux integrability.
    🔗
    To show that a function is Darboux integrable we must show that \(U(f) = L(f)\text{.}\) The difficulty is that computing \(U(f)\) and \(L(f)\) requires taking the supremum and infimum over all possible partitions. The next result allows us to show that \(U(f)= L(f)\) without needing to consider all possible partitions.
    🔗

    Theorem 69. Integrability Criteria.

    A bounded function is Darboux integrable on \([a,b]\) if and only if, for every \(\epsilon>0\text{,}\) there exists a partition \(P_{\epsilon}\) of \([a,b]\) such that
    \begin{equation*} U(f, P_{\epsilon}) - L(f, P_{\epsilon}) < \epsilon. \end{equation*}
    🔗
    🔗

    Proof.

    We first prove the reverse direction. Let \(\epsilon>0\) be arbitrary. Then there exists \(P_{\epsilon}\) such that \(U(f,P_{\epsilon}) - L(f,P_{\epsilon}) < \epsilon\text{.}\)
    🔗
    Hence
    \begin{equation*} U(f)-L(f) \leq U(f,P_{\epsilon}) - L(f,P_{\epsilon}) < \epsilon \end{equation*}
    🔗
    Since \(\epsilon>0\) is arbitrary this means that \(U(f) = L(f)\) and thus \(f\) is Darboux integrable.
    🔗
    To prove the forward direction we will use a triangle inequality argument. However since \(U(f) \geq L(f)\) we will not need absolute values.
    🔗
    Because \(U(f)\) is the supremum of all upper sums, given \(\epsilon > 0\) there exists a partition \(P_1\) such that \(U(f,P_1) < U(f) + \frac{\epsilon}{2}\)
    🔗
    Likewise, there is a partition \(P_2\) such that \(L(f,P_2) > L(f) - \frac{\epsilon}{2}\text{.}\)
    🔗
    Let \(P_{\epsilon} = P_1 \cup P_2\) which is a refinement of both \(P_1\) and \(P_2\) and recall that since \(f\) is integrable \(U(f) = L(f)\text{.}\)
    🔗
    Hence
    \begin{align*} U(f,P_{\epsilon}) - L(f,P_{\epsilon}) \amp \leq U(f,P_1) - L(f,P_2)\\ \amp < \left(U(f)+ \frac{\epsilon}{2} \right) - \left( L(f) - \frac{\epsilon}{2} \right)\\ \amp = \frac{\epsilon}{2} + \frac{\epsilon}{2} = \epsilon. \end{align*}
    🔗
    🔗
    Example: Show that \(f(x) = 2x\) is Darboux integrable on the interval \([0,3]\)
    🔗

    Theorem 70.

    If \(f\) is continuous on \([a,b]\) then it is Darboux integrable on \([a,b]\text{.}\)
    🔗
    🔗

    Proof.

    Since \(f\) is continuous on a compact set, it must be uniformly continuous on \([a,b]\text{.}\)
    🔗
    Let \(\epsilon>0\) be arbitrarily chosen. Then there exists \(\delta>0\) such that \(|f(y)-f(z)| < \frac{\epsilon}{b-a}\) whenever \(|y-z| < \delta\text{.}\)
    🔗
    Now let \(P\) be a partition of \([a,b]\) where \(x_k - x_{k-1} < \delta\) on every subinterval \([x_{k-1}, x_k]\) of \([a,b]\text{.}\)
    🔗
    On the subinterval \([x_{k-1},x_k]\) the supremum \(M_k\) occurs at some point \(y_k\) and the infimum \(m_k\) occurs at some point \(z_k\text{.}\)
    🔗
    Since \(|y_k-z_k| < \delta\text{,}\) \(M_k-m_k < \frac{\epsilon}{b-a}\text{.}\)
    🔗
    Thus
    \begin{align*} U(f,P) - L(f,P) \amp = \sum_{k=1}^n M_k (x_k-x_{k-1}) - \sum_{k=1}^n m_k (x_k-x_{k-1})\\ \amp = \sum_{k=1}^n (M_k-m_k) (x_k-x_{k-1})\\ \amp < \frac{\epsilon}{b-a} \sum_{k=1}^n (x_k-x_{k-1})\\ \amp = \frac{\epsilon}{b-a} (x_n-x_1) = \frac{\epsilon}{b-a} (b-a) = \epsilon \end{align*}
    🔗
    Hence \(f\) is Darboux integrable by the Integrability Criterion.
