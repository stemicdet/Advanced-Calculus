    Section Week 13: Lebesgue’s Criterion for Riemann Integrability (7.6)

    Subsection Riemann Integrability and Continuity

    In our study of the Darboux (equivalently Riemann) integral we have proven that the following types of functions are always Darboux integrable.
    1. Continuous functions
      🔗
      🔗
    2. Functions with a finite number of discontinuities
      🔗
      🔗
    🔗
    We also saw that a function with an infinite number of discontinuities may or may not be integrable. Now we will further investigate when we can integrate a function with infinitely many discontinuities.
    🔗
    Recall Thomae’s function.
    \begin{equation*} t(x) = \begin{cases} \hfill 1 \amp \text{ if $x =0$} \\ \hfill 1/n \amp \text{ if $x = \frac{m}{n} \in \mathbf{Q} \setminus \{0\}$ is in lowest terms with } \\ \hfill 0 \hfill \amp \text{ if $x \notin \mathbf{Q}$} \end{cases} \end{equation*}
    🔗
    Thomae’s function is continuous on the set of irrational numbers and discontinuous on the set of rational numbers. Thus it is discontinuous on a countably infinite set.
    🔗

    Claim 80.

    \(t(x)\) is Darboux integrable on \([0,1]\) and \(\displaystyle \int_0^1 t(x) dx = 0\)
    🔗
    🔗

    Proof.

    We will first show that \(L(t,P) = 0\) for any partition \(P\) of \([0,1]\text{.}\)
    🔗
    Let \(P\) be a partition of \([0,1]\text{.}\) Since the irrational numbers are dense in the reals, any subinterval of \(P\) contains an irrational point and since \(t(x) = 0\) on the irrationals \(L(t,P) = 0\text{.}\)
    🔗
    Now we will show that for an arbitrary \(\epsilon> 0\text{,}\) we can find a partition \(P\) such that \(U(t,P) < \epsilon\text{.}\)
    🔗
    Let \(N\) be a natural number so that \(\frac{1}{n} < \frac{\epsilon}{2}\) for all \(n > N\text{.}\) This means that \(N > \frac{2}{\epsilon}\text{.}\)
    🔗
    Let \(\{r_n\}\) be a list of all rational numbers in \((0,1)\) whose denominator is less than \(N\text{.}\) Note that this is a finite set of numbers.
    🔗
    We will construct the partition \(P\) by specifying the subintervals instead of the points of the partition. Let \(r_1\) be the first number on the list and construct a subinterval centered at \(r_1\) contained in \([0,1]\) with width at most \(\frac{\epsilon}{4}\text{.}\)
    🔗
    Now construct a second subinterval centered at \(r_2\) that is contained in \([0,1]\text{,}\) doesn’t intersect the first subinterval and has width at most \(\frac{\epsilon}{8}\text{.}\)
    🔗
    We repeat this process until we exhaust all the points on the list. For \(r_n\text{,}\) construct a subinterval centered at \(r_n\) contained in \([0,1]\) that doesn’t intersect any of the previously constructed subintervals and that has width at most \(\frac{\epsilon}{2^{n+1}}\)
    🔗
    Our partition \(P\) consists of the endpoints of these subintervals. Note that in addition to the subintervals that we constructed, \(P\) also has other subintervals that make up the missing space in \([0,1]\) not part of the subintervals that we constructed.
    🔗
    We now show that \(U(t,P) < \epsilon\text{.}\) First we consider the upper sum over the intervals centered at the \(r_n\)’s in the list.
    🔗
    Since \(0 < t(x) < 1\) for all \(x\) it follows that the upper sum over the intervals centered at the \(r_n\)’s is less than
    \begin{equation*} \frac{\epsilon}{4} + \frac{\epsilon}{8} + \frac{\epsilon}{16} + \ldots = \frac{\epsilon}{2}. \end{equation*}
    🔗
    Now we will bound the upper sum over the other subintervals of the partition. Note that these subintervals contain none of the rational numbers with denominator less than \(N\text{.}\)
    🔗
    This means that on these subintervals \(t(x) \leq \frac{1}{N+1} < \frac{\epsilon}{2}\) by our choice of \(N\text{.}\)
    🔗
    Since the total width of all the subintervals is less than \(1\text{,}\) this means that the upper sum over these subintervals is less than \(\frac{\epsilon}{2} \cdot 1 = \frac{\epsilon}{2}\text{.}\)
    🔗
    Combining these results gives that \(U(t,P) < \epsilon\text{.}\)
    🔗
    Since \(L(t,P) = 0\) for all partitions \(P\) and for every \(\epsilon > 0\) we can construct a partition \(P\) with \(U(t,P) < \epsilon\text{,}\) it follows that Thomae’s function is Darboux integrable and
    \begin{equation*} \int_0^1 t(x) dx = 0. \end{equation*}
    🔗
    🔗
    This shows that a function with a countable infinite number of discontinuities can be Darboux integrable.
    🔗
    🔗

    Subsection The Cantor Set

    Our next example involves the Cantor set which we studied last semester.
    🔗
    The Cantor set is constructed by starting with the interval \([0,1]\) and removing the middle third from this interval. This gives us step 1 of the construction of the Cantor set \(C_1 = [0,\frac{1}{3}] \cup [\frac{2}{3}, 1]\)
    🔗
    Next we remove the middle thirds from the remaining intervals to obtain \(C_2 = [0, \frac{1}{9}] \cup [\frac{2}{9}, \frac{1}{3}] \cup [\frac{2}{3}, \frac{7}{9}] \cup [\frac{8}{9}, 1]\)
    🔗
    The Cantor set is obtained by iterating this construction infinitely many times.
    🔗
    We review some of the properties of the Cantor set.
    1. The Cantor set contains an uncountably infinite number of points but has no length.
      🔗
      🔗
    2. The Cantor set is a closed nowhere dense set, which means its compliment is dense in \([0,1]\text{.}\)
      🔗
      🔗
    🔗
    Let \(C\) denote the Cantor set and define
    \begin{equation*} h(x) = \begin{cases} \hfill 1 \amp \text{ if $x \in C$} \\ \hfill 0 \amp \text{ if $x \notin C$} \end{cases} \end{equation*}
    🔗
    We will show that \(h(x)\) is discontinuous at each point in \(C\) and continuous at each point outside \(C\text{.}\) Thus \(h(x)\) on an uncountably infinite set of points.
    🔗
    Suppose \(x \in C\text{.}\) Because the compliment of the Cantor set is dense in \([0,1]\) we can find a sequence \(\{x_n\} \notin C\) that converges to \(x\text{.}\)
    🔗
    Thus \(h(x_n) \to 0\) as \(n\to \infty\) but \(h(x) = 1\text{.}\) Thus by the sequential definition of continuity, \(h\) is not continuous at \(x\text{.}\)
    🔗
    Now suppose \(x \notin C\text{.}\) Since the Cantor set is closed, the compliment of the Cantor set is an open set.
    🔗
    Thus there exists a \(\delta\)-neighborhood \(V_{\delta}(x)\) of \(x\) that is entirely contained in the compliment of the Cantor set. Hence if \(y \in V_{\delta}(x)\) then
    \begin{equation*} |h(y)-h(x)| = 0 - 0 = 0 \end{equation*}
    🔗
    Thus for any \(\epsilon\)-neighborhood \(V_{\epsilon}(h(x))\) of \(h(x)\text{,}\) if \(y \in V_{\delta}(x)\) then \(f(y) \in V_{\epsilon}(h(x))\) and thus \(h\) is continuous at \(x\) by the topological characterization of continuity.
    🔗

    Claim 81.

    \(h(x)\) is Darboux integrable on \([0,1]\) and \(\displaystyle \int_0^1 h(x)dx = 0\text{.}\)
    🔗
    🔗

    Proof.

    We will first show that \(L(h,P) = 0\) for every partition \(P\) of \([0,1]\text{.}\)
    🔗
    Let \(P\) be a partition of \([0,1]\text{.}\) Since the compliment of the Cantor set is dense in \([0,1]\text{,}\) every subinterval of \(P\) contains a point in the compliment of the Cantor set and since \(h(x) = 0\) outside of \(C\text{,}\) \(L(h,P) = 0\text{.}\)
    🔗
    Now we will show that for an arbitrary \(\epsilon > 0\) we can find a partition \(P\) of \([0,1]\) such that \(U(h,P) < \epsilon\text{.}\)
    🔗
    The construction of the Cantor set gives us a natural way to choose a partition. Let \(C_n\) be the \(n\)th stage in constructing the Cantor set. Then \(C_n\) contains \(2^n\) intervals each of which have length \(\left(\frac{1}{3}\right)^n\text{.}\)
    🔗
    Hence the total length of \(C_n\) is
    \begin{equation*} 2^n \cdot \left(\frac{1}{3}\right)^n = \left( \frac{2}{3}\right)^n. \end{equation*}
    🔗
    Note that every point of the Cantor set is included in \(C_n\text{.}\) Since \(\lim_{n\to \infty} \left( \frac{2}{3}\right)^n = 0\text{,}\) we can find a natural number \(N\) large enough so that \(\left( \frac{2}{3}\right)^N < \epsilon\) and thus the total length of \(C_N\) is less than \(\epsilon\text{.}\)
    🔗
    Let \(P\) be the partition given by the endpoints of \(C_N\text{.}\) This partitions \([0,1]\) into the intervals that make up \(C_N\) and the gaps in the middle of those intervals.
    🔗
    The maximum value of \(h(x)\) on the intervals that make up \(C_N\) is \(1\) and thus the upper sum over those intervals is
    \begin{equation*} 1 \cdot \left( \frac{2}{3}\right)^N = \left( \frac{2}{3}\right)^N < \epsilon \end{equation*}
    🔗
    Since the gaps are outside the Cantor set, the upper sum over the gaps is \(0\text{.}\) Hence \(U(h,P) < \epsilon\text{.}\)
    🔗
    The above proof almost works, but there is one slight flaw with it. Consider the subinterval of the leftmost gap. This is the subinterval \([\frac{1}{3^N}, \frac{2}{3^N}]\text{.}\) Since this subinterval contains the endpoints, which are part of the Cantor set, the maximum value of the subinterval is \(1\text{,}\) so the upper sum on this subinterval is \(\frac{1}{3^N}\)
    🔗
    The same is true for the subintervals over all the gaps, so the upper sum over the gaps is not \(0\text{.}\)
    🔗
    The problem is with the endpoints since every other point in the subintervals over the gaps is in the compliment of the Cantor set.
    🔗
    Thus we need to slightly modify our original partition so that the endpoints of \(C_N\) are not part of the subintervals that contains the gaps.
    🔗
    We can achieve this by slightly widening the partition so that each piece of \(C_N\) is contained in the interior of a subinterval.
    🔗
    Let \(\delta > 0\) to be determined later, and let \(P'\) be the partition \(\{0, \frac{1}{3^N} + \frac{\delta}{2}, \frac{2}{3^N} - \frac{\delta}{2}, \frac{3}{3^N} + \frac{\delta}{2}, \ldots, \frac{3^N-1}{3^N} - \frac{\delta}{2}, 1\} \)
    🔗
    \(P'\) is made by widening each subinterval of \(P\) by \(\frac{\delta}{2}\) on each side, except for the 1st and last subinterval since those have to contain \(0\) and \(1\) as endpoints.
    🔗
    Now that the subintervals over the gaps only contain points in the compliment of the Cantor set, the upper sum over those subintervals is \(0\text{.}\)
    🔗
    We now recalculate the upper sum over those subintervals that contain the pieces of \(C_N\text{.}\) This time the length of each subinterval is \(\left(\frac{1}{3}\right)^N + \delta\text{,}\) except for the first and last which have length \(\left(\frac{1}{3}\right)^N + \frac{\delta}{2}\text{.}\)
    🔗
    Since there are \(2^N\) total subintervals, the total length of these subintervals is less than
    \begin{equation*} 2^N \cdot \left( \left(\frac{1}{3}\right)^N + \delta \right) = \left(\frac{2}{3}\right)^N + 2^N \delta \end{equation*}
    🔗
    By choosing \(\delta = \left(\frac{1}{3}\right)^N\text{,}\) the total length of the subintervals will be less than \(2 \left(\frac{2}{3}\right)^N\text{.}\)
    🔗
    Thus the upper sum over these intervals is
    \begin{equation*} 1 \cdot 2 \left(\frac{2}{3}\right)^N < 2 \epsilon \end{equation*}
    🔗
    Since the upper sum over the gaps is \(0\text{,}\) \(U(h,P') < 2 \epsilon\)
    🔗
    Since \(\epsilon>0\) can be made arbitrarily small, so can \(2 \epsilon\text{.}\)
    🔗
    Because \(L(h,P) = 0\) for all partitions \(P\) and for every \(\epsilon > 0\) we can construct a partition \(P_{\epsilon}\) with \(U(h,P_{\epsilon}) < \epsilon\text{,}\) it follows that \(h(x)\) is Darboux integrable and
    \begin{equation*} \int_0^1 h(x) dx = 0. \end{equation*}
    🔗
    🔗
    This shows that a function with an uncountable number of discontinuities can still be Darboux integrable.
    🔗
    🔗

    Subsection Lebesgue’s Theorem

    Our two examples show that a function with infinitely many discontinuities can still be Darboux integrable; however, not every function with infinitely many discontinuities is Darboux integrable. Dirichlet’s function is everywhere discontinuous and is not Darboux integrable.
    🔗
    The key is that the set of discontinuities of the previous two examples is still small in the following sense. The set of rational numbers and the Cantor set both have zero length.
    🔗
    Informally we say that a subset of \(\mathbf{R}\) with zero length is a set of measure zero.
    🔗

    Theorem 82. Lebesgue’s Theorem.

    Let \(f\) be a bounded function on the interval \([a,b]\text{.}\) Then \(f\) is Darboux integrable if and only if the set of points where \(f\) is not continuous is a set of measure zero.
    🔗
    🔗
    Dirichlet’s function is not Darboux integrable because it is discontinuous everywhere, and thus the set of points where it is not continuous is not a set of measure zero.
    🔗
    🔗

    Subsection A Non-integrable Derivative

    Informally part 1 of the FTC states that the definite integral of a derivative is the net change of the original function. However, in the statement of the FTC we needed to assume that the derivative is an integrable function. This is because there are derivatives that cannot be integrated.
    🔗
    More precisely, there exists a function \(f(x)\) which is not Darboux integrable and a continuous function \(F(x)\) such that \(F'(x) = f(x)\)
    🔗
    The key idea is that we want to construct a derivative whose set of discontinuities is not a set of measure zero.
    🔗
    We currently only know one example of a function whose derivative contains a discontinuity: Darboux’s function
    \begin{equation*} g(x) = \begin{cases} \hfill x^2 \sin(1/x) \amp \text{ if $x \neq 0$} \\ \hfill 0 \hfill \amp \text{ if $x =0 $} \end{cases} \end{equation*}
    is a continuous function whose derivative has a single discontinuity at \(x=0\text{.}\)
    🔗
    We need to modify Darboux’s function so that the set of discontinuities of its derivative is not a set of measure zero.
    🔗
    Let’s first focus on what the set of discontinuities is going to look like.
    🔗
    The Cantor set is a set with measure zero because it has length zero, but by modifying the construction of the Cantor set, we can construct a fat Cantor set with the same properties as the Cantor set, but which is not a set of measure zero.
    🔗
    In the construction of the Cantor set, we start with the interval \([0,1]\) remove the middle third of each interval at each iteration.
    🔗
    This means that at the \(n\)th stage we remove \(2^{n-1}\) intervals of length \(\frac{1}{3^n}\text{.}\)
    🔗
    Thus the total amount removed from \([0,1]\) has length
    \begin{equation*} \sum_{n=1}^{\infty} \frac{2^{n-1}}{3^n} = \sum_{n=0}^{\infty} \frac{1}{3} \left(\frac{2}{3} \right)^n = 1 \end{equation*}
    🔗
    To construct the fat Cantor set, at the \(n\)th stage we will remove \(2^{n-1}\) intervals of length \(\frac{1}{3^{n+1}}\text{.}\)
    🔗
    The total length removed from \([0,1]\) has length
    \begin{equation*} \sum_{n=1}^{\infty} \frac{2^{n-1}}{3^{n+1}} = \sum_{n=0}^{\infty} \frac{1}{9} \left(\frac{2}{3} \right)^n = \frac{1}{3} \end{equation*}
    🔗
    This means that the fat Cantor set has length \(\frac{2}{3}\) and thus is not a set of measure zero.
    🔗
    The reason that the derivative of Darboux’s function is not continuous at \(0\) is because the function oscillates wildly as it approaches \(0\text{.}\)
    🔗
    Our plan to construct a function with a non-integrable derivative is to modify Darboux’s function so that it has the same oscillatory behavior at every point of the fat Cantor set.
    🔗
    This will give us a function whose derivative is discontinuous on the fat Cantor set and thus can’t be Darboux integrable since the fat Cantor set does not have measure zero.
    🔗
    Let \(C_0 = [0,1]\) and let \(C_n\) denote the \(n\)th stage of the construction of the fat Cantor set.
    🔗
    Because at the \(n\)th stage we remove intervals of width \(\frac{1}{3^{n+1}}\) instead of \(\frac{1}{3^n}\text{,}\) \(C_1 = [0,\frac{4}{9}] \cup [\frac{5}{9}, 1]\text{.}\)
    🔗
    Similarly,
    \begin{align*} C_2 \amp= \left[0, \frac{5.5}{27}\right] \cup \left[\frac{6.5}{27}, \frac{4}{9}\right]] \cup \left[\frac{5}{9}, \frac{20.5}{27}\right] \cup \left[\frac{21.5}{27}, 1\right] \\ \amp= \left[0, \frac{11}{54}\right] \cup \left[\frac{13}{54}, \frac{4}{9}\right] \cup \left[\frac{5}{9}, \frac{41}{54}\right] \cup \left[\frac{43}{54}, 1\right] \end{align*}
    🔗
    We will construct our function with a non-integrable derivative as the limit of a sequence of continuous functions.
    🔗
    Let \(f_0(x) = 0\) on \(C_0\text{.}\)
    🔗
    To define \(f_1(x)\text{,}\) first set \(f_1(x) = 0\) on \(C_1 = [0,\frac{4}{9}] \cup [\frac{5}{9}, 1]\text{.}\)
    🔗
    In the gap \((\frac{4}{9}, \frac{5}{9})\) we want to put copies of Darboux’s function \(g(x)\) oscillating towards the endpoints.
    🔗
    The mathematical formula for this looks like
    \begin{equation*} f_1(x) = \begin{cases} \hfill 0 \amp \text{ if $x \in [0, \frac{4}{9}]$} \\ \hfill g(x-\frac{4}{9}) \amp \text{ if $x$ is just to the right of $\frac{4}{9}$} \\ \hfill g(-x + \frac{5}{9}) \amp \text{ if $x$ is just to the left of $\frac{5}{9}$} \\ \hfill 0 \amp \text{ if $x \in [ \frac{5}{9}, 1]$} \end{cases} \end{equation*}
    🔗
    We still need to define what happens in the middle of the function.
    🔗
    The idea is that we connect the two oscillating pieces together so that \(f_1(x)\) will be differentiable and so that \(|f_1(x)| \leq (x-\frac{4}{9})^2\) and \(|f_1(x)| \leq (-x+\frac{5}{9})^2\text{.}\)
    🔗
    \(f_1(x)\) is a continuous function whose derivative is discontinuous at \(x = \frac{4}{9}\) and \(x= \frac{5}{9}\text{.}\)
    🔗
    To define \(f_2(x)\) we do the same thing as we did with \(f_1(x)\) only this time we use the open intervals \((\frac{11}{54}, \frac{13}{54})\) and \((\frac{41}{54}, \frac{43}{54})\)
    🔗
    \(f_2(x)\) is a continuous function whose derivative is discontinuous at \(x= \frac{11}{54}\text{,}\) \(x= \frac{13}{54}\text{,}\) \(x= \frac{41}{54}\text{,}\) and \(x= \frac{43}{54}\text{.}\)
    🔗
    Continuing in this way we construct a sequence of functions \(f_1\text{,}\) \(f_2\text{,}\) \(f_3 \ldots\text{.}\) Let \(f(x) = \displaystyle \lim_{n\to \infty} f_n(x)\text{.}\)
    🔗
    \(f(x)\) is a continuous function whose derivative is discontinuous at every point of the fat Cantor set.
    🔗
    Thus \(f'(x)\) is not a Darboux integrable function, despite the fact that it has an antiderivative in \(f(x)\text{.}\)
